# 💅 NailBook

Modern online appointment booking system for nail technicians.

## ✨ Features

- Online booking
- Customer accounts
- Admin dashboard
- Calendar
- Email reminders
- Statistics

## 🛠 Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- Supabase
- Auth.js
- Resend
